1. Build
   
Download or clone this git project, then use ChimeraX command console to build:

devel build /path/to/chimerax-chat

2. Install

Use ChimeraX command console to install:

toolshed install /path/to/chimerax-chat/dist/ChimeraX_Chat-0.1.0-py3-none-any.whl

3. Use
   
Lanuch the Chat box from "Tool" menu, or run command:

ui tool show "AI Chat"

First time you run the chat box, it will prompt you to enter your api key. You can get a free Google AI Studio api key from https://aistudio.google.com/app/api-keys
