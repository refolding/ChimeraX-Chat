Install
1. Download dist/chimerax_chat-0.1.0-py3-none-any.whl
2. in ChimeraX, run command:
	toolshed install /Path/to/ChimeraX_Chat-0.1.0-py3-none-any.whl
3. Lauch the Chat box from "Tool" manu, or run command:
	ui tool show "AI Chat"

First time your run the chat box, it will prompt you to enter your api key. You can get a free Google AI Studio api key from https://aistudio.google.com/app/api-keys
