1. Build

Download or clone this Git project, then use the ChimeraX command console to build:

devel build /path/to/chimerax-chat

2. Install

Use the ChimeraX command console to install:

toolshed install /path/to/chimerax-chat/dist/ChimeraX_Chat-0.1.0-py3-none-any.whl

3. Use

Launch the chat box from the "Tools" menu, or run:

ui tool show "AI Chat"

The first time you run the chat box, it will prompt you to enter your API key. You can get a free Google AI Studio API key from:
https://aistudio.google.com/app/api-keys
